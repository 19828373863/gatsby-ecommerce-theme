2023年客户对潜在风险客户贷款确认不良并及时进行处置导致拨备覆盖率下降。

1.$$资本充足率=\frac{总资本 - 对应资本扣除项 }{风险加权资产 } \times 100 %$$——摘录自商业银行资本管理办法。该数值直接取 G40资本充足率汇总表C35格填列的数据。

2.$$一级资本充足率 =\frac{一级资本 - 对应资本扣除项 }{风险加权资产 } \times 100$$。——摘录自商业银行资本管理办法。该数值直接取 G40资本充足率汇总表C36格填列的数据。

3.$$核心一级资本充足率=\frac{核心一级资本 - 对应资本扣除项 }{ 风险加权资产 }\times 100$$ ——摘录自商业银行资本管理办法。该数值直接取 G40资本充足率汇总表C37格填列的数据。

4.准备金充足程度指标包括资产损失准备充足率和贷款损失准备充足率。资产损失准备充足率为一级指标，为信用风险资产实际计提准备与应提准备之比，不应低于100%；贷款损失准备充足率为贷款实际计提准备与应提准备之比，不应低于100%，属二级指标。——摘录自商业银行风险监管核心指标（试行）。该数据取S3a00-201-境内汇总数据-人民币.xls表内数据，贷款损失准备应提计算公式如下：贷款总额的1%，关注类的2%，次级的25%，可疑的50%，损失的100%；计提的准备金为C51数据。=C51/(C400.01+F400.02+H400.25+I400.5+J40)。

5.核心资本比例=核心资本净额/资本净额。取G40表

6.单一借款人授信比例在G1403最大十家客户贷款情况表中。

8.流动性比例为流动性资产余额与流动性负债余额之比，衡量商业银行流动性的总体水平，不应低于25%。——摘录自商业银行风险监管核心指标（试行）。取G22流动性比例监测表C30单元格数据**245.65%**填列。

9.$$ 杠杆率 =\frac{一级资本-一级资本扣除项 }{调整后表内外资产余额 } \times 100 %$$。杠杆率，是指商业银行持有的、符合本办法规定的一级资本净额与调整后表内外资产余额之间的比率。——商业银行资本管理办法。该数值直接根据G44杠杆率情况表 C29=22.1172%单元格数据填列。

10.拨备覆盖率为贷款损失准备与不良贷款余额之比。——摘录自商业银行贷款损失准备管理办法。贷款损失准备取**G4A-1(a)贷款损失准备情况表（权重法）**中C5:实际计提的贷款损失准备余额,不良贷款余额取SUM(C17:C19).计算得789.00%

11.不良贷款率为不良贷款与贷款总额之比——摘录自商业银行风险监管核心指标（试行）。取**G4A-1(a)贷款损失准备情况表（权重法）**中=SUM(C17:C19)/C14，得到0.63%。
