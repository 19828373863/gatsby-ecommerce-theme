IPMT 函数

一、函数介绍

IPMT 函数用于基于等额分期付款和固定利率，计算为某项投资指定期数所支付的利息。

二、函数解读

函数公式：=IPMT(利率, 期数, 总期数, 现值, [期值], [期末或期初])

参数释义：

利率 （必填）：利息率。

期数（必填）：用于计算其本金数额的期数，必须介于 1 到总期数之间。

总期数（必填）：需要支付的次数。

现值（必填）：现值，或一系列未来付款的当前值的累积和，也称为本金。

期值（选填）：为未来值，或在最后一次付款后希望得到的现金余额，如果省略此参数，则假设其值为零，也就是一笔贷款的未来值为零。

期末或期初（选填）：指示付款时间是在各期的期末 (0) 还是期初 (1)。

举个例子：

=IPMT(1%, 1, 360, 100000, 30000, 0)

注意事项：

利率和总期数的单位需要统一。比如，同样是四年期年利率为 12% 的贷款，如果按月支付，利率应为 12%/12，总期数应为 4*12；如果按年支付，利率应为 12%，总期数为 4。 对于所有参数，支出的款项，如银行存款，表示为负数；收入的款项，如股息收入，表示为正数。 三、操作步骤使用

IPMT 函数选中单元格，在单元格内输入 =IPMT在单元格中输入公式参数，并计算两种不同付款周期下对应的利息。=IPMT(B2/12,B3,B4*12,B5) =IPMT(B2,3,B4,B5)按回车键，单元格中将显示判断的结果

PMT 函数

一、函数介绍PMT 函数常用于返回年金的定期支付金额。

二、函数解读函数公式：=PMT(利率, 总期数, 现值, [期值], [期末或期初])

参数释义： 利率（必填）：贷款利率。

总期数（必填）：该项贷款的付款总数。

现值（必填）：现值，或一系列未来付款的当前值的累积和，也称为本金。

[期值]（选填）：为未来值，或在最后一次付款后希望得到的现金余额，如果省略此参数，则假设其值为零，也就是一笔贷款的未来值为零。

[期末或期初]（选填）：数字 0 或 1 付款时间是在期初还是期末。

举个例子：=PMT(1%, 360, 100000, 0, 0)

注意事项： PMT 函数返回的支付款项包括本金和利息，但不包括税款、保留支付或某些与贷款有关的费用。应确认所指定的利率和总期数单位的一致性。比如，同样是四年期年利率为 12% 的贷款，如果按月支付，利率应为 12%/12 （“/” 的意思是除以），总期数应为 4*12；如果按年支付，利率应为 12%，总期数为 4。如果要计算贷款期间的支付总额，请用 PMT 函数返回值乘以总期数。

三、操作步骤使用 PMT 函数选中单元格，在单元格内输入 = PMT在单元格中输入公式参数 =PMT(B2/12,B3,B4) 按回车键，单元格中将显示判断的结果 -8652.67461

PV 函数

一、函数介绍

PV 函数常用来返回投资的现值。注：现值的英文是 present value，表示在给定的利率水平下，未来的资金折线到现在时刻的价值，是资金时间价值的逆过程。

二、函数解读函数公式：=PV(利率,总期数,付款金额, [期值], [期末或期初])

参数释义：

利率（必填）：各期的贷款利率。如果按 12% 的年利率借入一笔贷款来购买房屋，并按月偿还贷款，则月利率为 12%/12（即 1%）。

总期数（必填）：该项贷款的付款总数。

付款金额（必填）：每期的付款金额，其数值在整个年金期间保持不变。通常 pmt 包括本金和利息，但不包括其他费用及税款。

[期值] （选填）：为未来值，或在最后一次付款后希望得到的现金余额，如果省略此参数，则假设其值为零，也就是一笔贷款的未来值为零。

[期末或期初] （选填）：数字 0 或 1 付款时间是在期初还是期末。

举个例子：=PV(2%, 12, 100, 0, 0)

注意事项： Lark 表格公式里的利率，也就是第一个参数，不可以为 0 哦！应确认所指定的利率和总期数单位的一致性。比如，同样是四年期年利率为 12% 的贷款，如果按月支付，利率应为 12%/12 （“/” 的意思是除以），总期数应为 4*12；如果按年支付，利率应为 12%，总期数为 4。

三、操作步骤

  1. 选中单元格，在单元格内输入 =PV

  2. 在单元格中输入公式参数 =PV(2%, 12, 100, 0, 0)

  3. 按回车键，单元格中将显示判断的结果 -1057.534122

四、场景实践

  • 投资必备：使用 PV 函数快速计算投资现值

资金的价值是变动的，根据各种金融环境而变化，由此有了现值和未来值的概念。在投资时，为了计算现值，就可以用 PV 函数，属于需要的参数，就可以一步得到结果。

  • 下图中所用的公式：=PV(B2/12,B3*12,B4)

  • 公式参数解读：

    • 下图例子中，由于贷款利率是年利率，因此需要进行单位统一，换算成月，因此用 B2/12 换算成月贷款利率作为参数。
    • 由于年限 16 也是单位为年，因此需要将 16*12 换算成月为单位作为总期数。
    • B4 单元格里的参数已经是按每月支付，因此无需进行单位换算，直接点击单元格作为参数即可。
    • 点击回车键得出结果。

RATE 函数

一、函数介绍

RATE 函数指基于等额分期付款的方式，返回某项投资或贷款的实际利率。

RATE 函数通过迭代法计算得出结果，并且可能无解或有多个解。 如果在进行 20 次迭代计算后，RATE 函数的相邻两次结果没有收敛于 0.0000001，RATE函数将返回错误值 #NUM!。

注：迭代法是通过从一个初始估计出发寻找一系列近似解来解决问题（一般是解方程或者方程组）的数学过程，为实现这一过程所使用的方法统称。

二、函数解读

  • 函数公式：=RATE(期数, 每期的支付额, 现值, [期值], [期末或期初], [收益率估值])

  • 参数释义：

    • 期数（必填）：需要支付的次数。
    • 每期的支付额（必填）：每期要支付的金额。
    • 现值（必填）：现值，或一系列未来付款的当前值的累积和，也称为本金。
    • 期值（选填）：完成最后一次支付后的剩余期值。
    • 期末或期初（选填）：指示付款时间是在各期的期末 (0) 还是期初 (1)。
    • 收益率估值（选填）：收益率的估算值。

  • 举个例子：=RATE(12, -100, 400, 0, 0, 1%)

  • 注意事项：

    • RATE 函数假设在每个期末付款。

三、操作步骤

使用 RATE 函数

  1. 选中单元格，在单元格内输入 =RATE

  2. 在单元格中输入公式参数 =RATE(12, -100, 400, 0, 0, 1%)

  3. 按回车键，单元格中将显示判断的结果 23%

删除 RATE 函数

选中应用了 RATE 函数的单元格，按删除键，清空单元格内的函数公式。

四、场景实践

  • 财务必备：使用 RATE 函数快速求利率

当想要求某项投资或贷款的实际利率时，可以按照等额分期付款的方式，使用 RATE 函数，输入所需的参数一步求利率。

  • 下图中所用的公式：

    • =RATE(B2*12,B3,B4)
    • =B6*12

  • 公式参数解读：

    • 下图例子中贷款期限 4 的单位是年，因此当按月支付时需要乘 12
    • 第二个参数是每期的支付额，也就是 200，但由于是开支，因此前面加上了负号
    • 第三个参数是年金的现值，也就是 8000
    • 以上三个参数是必填的，下图例子我们省略了期值、期末或期初、收益率估值这三个选填值，可以根据具体情况进行添加。
    • 计算贷款的年利率时可以在贷款月利率的结果上乘 12，从而更加方便得得出计算结果

IRR 函数

一、函数介绍

IRR 函数通常用于基于一组定期现金流计算投资的内部回报率。

二、函数解读

  • 函数公式：= IRR(现金流数额, [收益率估值])

  • 参数释义：

    • 现金流数额（必填）其中含有投资相关收益或支出的数组或范围。
    • 收益率估值（选填）内部收益率的估算值。

  • 举个例子：=IRR(A2:A25, 1%)

  • 注意事项：

    • 函数 IRR 与函数 NPV 函数（净现值函数）的关系十分密切。函数 IRR 计算出的收益率即净现值为 0 时的利率。
    • 下面的公式显示了函数 NPV 和函数 IRR 的相互关系：
    • NPV(IRR(B1:B6),B1:B6) 等于 3.60E-08（在函数 IRR 计算的精度要求之中，数值 3.60E-08 可以当作 0 的有效值）。

三、操作步骤

使用 IRR 函数

  1. 选中单元格，在单元格内输入 =IRR

  2. 在单元格中输入公式参数 =IRR(B2:B5)

  3. 按回车键，单元格中将显示判断的结果 -46%

删除 IRR 函数

选中应用了 IRR 函数的单元格，按删除键，清空单元格内的函数公式。

四、场景实践

  • 财务必备：使用 IRR 函数快速计算投资内部收益率

每一年投资的内部收益率都会有所不同，为了方便快速计算收益率，可以使用 IRR 函数进行快速计算。

  • 下图中所用的公式：

    • =IRR(B2:B6)
    • =IRR(B2:B7)

  • 公式参数解读：想要计算四年后的内部收益率只需要将范围从“第一年的净收入”拉至“第四年的净收入”即可，五年后的内部收益率同理。
