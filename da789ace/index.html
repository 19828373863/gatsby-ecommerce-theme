市场组织结构

**中国人民银行(PBOC)：**银行间市场监管机构

中国银行间市场交易商协会（NAFMII）：银行间市场自律组织，履行自律监管职能。

全国银行间同业拆借中心(CFETS)：提供交易平台及交易后处理、信息、基准、监测等相关服务，提供shibor等数据，是交易中心，交易的前台。

**中央国债登记结算有限责任公司(CCDC)：**债券托管、结算等

**银行间市场清算所股份有限公司(SHCH)：**债券托管、结算、中央对手方(CCP)清算等

**人民银行清算总中心、商业银行：**提供资金清算服务

**跨境银行间支付清算有限责任公司(CIPS)：**人民币跨境支付清算服务

**交易成员：**各类金融机构、部分非金融机构及非法人产品

交易成员结构

存款类机构

包括：政策性银行、境内商业银行、农村商业银行、农村信用联社。

银行间货币市场

货币市场交易品种⭐⭐

质押式回购：占比最大，其中隔夜利率回购的质押式回购占比最多。有担保。

买断式回购：所有权是否转移是区分质押式和买断式的标志。交易较为繁琐。可以看作两次方向相反的债券买卖。

同业存单、券商短融：可以看作是债券。

交易方式：

**询价为主：**包括iDeal聊天/资金意向的转对话，适用于所有交易品种，便于控制对手方风险。

请求报价：适用于质押式回购，便于资金高效管理与合理分配

匿名点击（X-Repo）：适用于质押式回购，提高交易效率

**同业存单/券商短融：**交易方式同现券，包括询价、做市报价、请求报价、匿名点击等

同业拆借市场概述⭐⭐

**参与主体：**符合《同业拆借管理办法》要求的机构，可直接向交易中心提交联网材料（一站式平台【银行间本币市场开户系统】）

  • 根据《同业拆借管理办法》第六条，16类金融机构可以申请进入同业拆借市场，涵盖所有银行类金融机构和绝大部分非银行金融机构（详见政策法规培训教材）

  • 中国人民银行认定的其他机构目前包括境外人民币清算行等

**交易平台：**同业拆借交易必须在全国统一的拆借网络进行——全国银行间同业拆借中心本币交易系统。

  • 询价交易方式：对话报价（本方报价→对手方确认成交）、资金意向转对话（发送/查看资金意向→将意向报价转为对话报价→双方确认成交）

**主要功能：**各类金融机构融通短期资金、进行主动资产负债管理的有效途径。

同业拆借市场交易要素⭐⭐⭐

交易要素 含义
拆借方向 拆入或拆出
拆借期限 拆借天数(最长不超过360天)【（考题）不是一年，是360天】
拆借利率 拆入方付给拆出方的资金价格，以年利率表示，支持按市场基准加减点报价
拆借金额 拆借金额，最低为10万元，最小变动量为10万元
清算速度 达成交易到实际清算的天(工作日)数，有T+0(成交当日清算)和T+1(成交次日清算)两种
成交日 达成交易的日期
首次结算日 首次结算日=成交日+清算速度(遇节假日顺延到下一工作日),是资金从拆出方向拆入方发生实际转移的日期
到期还款日 到期还款日=首次结算日+拆借期限(遇节假日顺延到下一工作日),是资金从拆入方实际归还至拆出方的日期
实际占款天数 实际占款天数=到期还款日-首次结算日
应计利息 到期还款日拆入方还给拆出方的利息总额，计息基准为“实际天数/360”
到期还款金额 到期还款金额=首次结算金额+应计利息总额，是到期还款日拆入方还给拆出方的总金额
交易品种 拆借期限所属的统计区间，包括IBO001、IBO007、IBO014、IBO021、BO1M、IBO2M、IBO3M、IBO4M、IBO6M、IBO9M和IBO1Y共11个品种

IBO：interbankoffering

IBO001:拆借1天

IBO007：拆借2-7天

清算速度T+0：今日达成交易今日完成现金支付。

回购市场概述⭐⭐

**参与主体：**成员广泛，各类金融机构和非法人投资者皆可通过回购市场进行资金融通和头寸管理

  • 境外参与主体包括境外央行类（境外央行或货币当局、国际金融组织、主权财富基金），境外清算行，境外参加行。

**回购券种：**银行间债券市场的各类券种（包括外币债、SDR计价债券）均可用于回购，非公开定向发行债务融资工具(PPN)和资产支持证券（ABS）仅支持质押式回购

ABS、PPN只能做质押式回购，不能做买断式回购。

  • 2015年5月9日起，投资者可以自己发行的债券为标的资产进行债券回购交易（需进行信息披露，并符合比例要求）

**交易平台：**回购交易必须通过交易中心本币交易系统达成（中国人民银行公告【2013】第8号）

质押式回购

质押式回购交易要素⭐⭐⭐

交易要素 含义
交易方向 正回购或逆回购，即融入或融出资金
回购期限 回购天数(最长不超过365天)【考题，就是365，不区分平年闰年】
回购利率 正回购方付给逆回购方的资金价格，以年利率表示
债券代码/债券名称 质押债券代码/名称，单笔交易的质押券数量最多为15只，可跨托管机构
券面总额 质押债券的券面总额，最低为10万元，最小变动单位为10万元
折算比例 折算比例=实际融入金额/质押债券券面总额*100%
交易金额 正回购方实际融入金额
应计利息 到期日正回购方还给逆回购方的利息总额，计息基准为“实际天数/365”
到期结算金额 到期结算金额=交易金额+应计利息总额，是到期日正回购方返还的金额
清算速度 达成交易到实际清算的天(工作日)数，包括T+0和T+1
成交日 达成交易的日期
首次结算日 首次结算日=成交日+清算速度(遇节假日顺延到下一交易日),开始计息的日期
到期结算日 到期结算日=首次结算日+回购期限(遇节假日顺延到下一交易日),到期清算的日期
实际占款天数 实际占款天数=到期结算日-首次结算日
首次/到期结算方式 首次/到期结算时的资金支付和债券结算方式，均应采用券款对付(DVP)
交易品种 根据质押式回购交易期限，由系统自动计算并显示的回购期限所属的统计区间，包括：R001、R007、R014、R021、R1M、R2M、R3M、R4M、R6M、R9M、R1Y

质押券必须是银行间市场发行，且可流通

交易合约：R002指期限2天的合约,而不是某个时间区间。

【考题】质押式回购到期后可以展期

质押式回购交易方式⭐⭐

**询价：**适用于所有交易品种，便于控制对手方风险

  • 对话报价：本方报价→对手方确认成交

  • iDeal转对话：iDeal协商→对话报价→双方确认成交

  • 意向转对话：发送/查看资金意向→将意向报价转为对话报价→双方确认成交

**请求报价（RFQ）：**市场成员主动一对多询价

**匿名点击（X-Repo）：**匿名匹配+点击成交

  • 匿名发送报价→报价自动匹配/未匹配报价可供点击→交易关系/额度内成交

【题目】质押式回购交易方式有询价、请求报价和匿名点击。

请求报价（RFQ）⭐

**服务场景：**具有影响力的市场机构作为资金交易商，完善市场成员与资金交易商日常交易中的询量询价环节，有助于资金的高效管理、合理分配

参与角色：

  • 资金交易商：逆回购方，可维护本方质押RFQ白名单，收到请求报价后进行量价回复。

  • 市场成员：正回购方，可向已将本方列入RFQ白名单内的资金商发起RFQ，收到回复后进行确认、提券。

功能亮点：

  • 支持一对多发起请求、支持批量快捷回复、支持多产品分仓交易

  • 支持按市场基准加减点报价交易

  • 支持利用回购小助手进行风险管理

【考题】请求报价只能发送给做市机构，至少要发送给一家机构，需要填写券面总额。

【考题】请求报价默认有效时间：2小时

【考题】具名显示回复机构，成交量不能大于请求量，最优价格红色标签显示

【考题】回复请求报价时，需要填写净价或收益率，可以修改券面总额，可以拒绝请求报价

匿名点击（X-Repo）☆

**交易时间：**盘前交易时段（8:45-8:55)，连续交易时段（9:00-12:00,13:30-16:50）

交易合约：R001、R002、R003、R004、R007、R014、R021，R1M(28天）、R3M（91天）

**交易流程：**匿名限价订单匹配成交/未匹配订单供点击成交（双边限额条件内）+正回购方可预提质押券，或在成交后规定时间内提交质押券（30分钟内）

质押券要求：利率债（国债、央票、政金债）、地方政府债（目前为长三角、粤港澳大湾区主体）、同业存单（目前为系统重要性国股行），质押券折算率由CFETS计算并公布

交易规则：匿名限价，价格优先、时间优先，匹配后具名

**报价行情：**最优报价行情，深度档位行情，展示可成交报价量，点击价格达成成交

功能亮点：拓宽融资渠道、平补临时性头寸、获得低利率融资；深度档位行情“所见即所得”，提高交易效率，减少寻找对手方、询价等的时间成本。

盘前交易机制

  • 盘前交易时段（8:45-8:55），盘前报价机构可提交匿名限价订单进行报价，并在全市场范围内匿名成交；

  • 盘前交易时段其他参与机构不可提交报价，可对未成交报价匿名点击成交；

  • 盘前交易时段截止后，盘前报价机构可对未成交报价作修改、冻结或撤销；

  • 盘前交易时段，交易品种、交易流程、成交规则等与连续交易时段相同。

盘前报价机构

  • 根据市场影响力、交易规模、定价能力、内部风控机制等因素，为部分匿名回购业务参与机构开通盘前报价权限（目前35家）

盘前机制亮点

  • 提升早盘价格指示性，预判当日市场流动性松紧情况

买断式回购⭐⭐⭐

**交易品种：**买断式回购单券vs买断式回购多券

买断式回购单券 买断式回购多券
回购债券只数 单笔交易仅使用一只债券 单笔交易可使用多只债券
期间债券派息 逆回购方持有票息至到回购到期,相应的回购利率和到期结算金额计算会考虑票息占用情况 逆回购方在收到回购债券派息后，不晚于次一个营业日结束前支付给正回购方
主要功能 兼具融资融券双重属性 侧重于融资属性

**交易方式：**主要为询价，均适用于上述交易品种

  • 对话报价：本方报价→对手方确认成交

  • 资金意向转对话：发送/查看资金意向→将意向报价转为对话报价→双方确认成交

交易要素 含义

券面总额 回购债券面值的总量，最低为10万元，最小变动单位为10万

做市商制度意义⭐⭐

做市商制度是场外债券市场有效运行的重要基础性制度

  • 促进债券市场流动性提高

  • 促进债券市场价格发现

  • 维护市场稳定、规范市场发展

  • 降低发行人筹资成本

截至2023年底：92家做市商

现券市场交易方式⭐⭐

**询价：**适用于所有交易品种，便于控制对手方风险

  • 对话报价：本方报价→对手方确认成交

  • 意向报价：iDeal协商→对话报价→双方确认成交；直接发送对话报价→对手方确认成交

**做市报价：**做市商主动报价

  • 做市商发送点击成交报价→报价被点击、选择成交量→报价量内直接成交

  • 冰山订单：显示冰山券面总额的做市报价

**分层报价：**做市商发送针对特定群组报价

  • 做市商设定群组→针对不同群组发送分层报价

  • 用户查看私有行情→报价量内点击成交

  • 适合于拥有一定特定客户群体的做市商。配合做市报价，可以使做市报价更有针对性，进一步活跃市场。

指示性报价：具有指示性报价权限的做市商主动报价

  • 报价机构发送双向不可点击成交的报价→报价被点后转成RFQ成交

**请求报价（RFQ）：**非做市商主动询价

【题目】现券买卖请求报价的最终成交量可以大于最初的请求量。

**匿名点击（X-Bond）：**匿名匹配+点击成交

匿名发送报价→报价自动匹配/未匹配报价可供点击→授信范围内成交

  • 限价订单——参与机构提交的限定价格的订单→与优于或等于该价格的反方向订单匹配成交

  • 市价订单——参与机构提交的限定方向（无价格）的订单→按照市场价格、与反方向的订单匹配成交

  • 弹性订单——参与机构提交设置市场不可见底价的订单→与优于或等于底价价格的反方向订单匹配成交

**匿名拍卖（X-Auction）：**申报+双向报价+集中竞价

  • 预申报：提交拍卖意向

  • 申报日：在可申报债券列表范围内，申报拍卖债券

  • 拍卖日：双向报价——在一定价差范围内进行双向报价。用于定价格区间，不直接成交

集中竞价——用户发送限价订单→价格优先、时间优先、最大匹配量原则确定本交易场次的成交

匿名询价（X-Bargain）

  • 创新询价模式：匿名+明盘/暗盘+同时出价

  • 有限轮次价格博弈，确保交易效率

  • 引入做市商满足合规，支持交易后分仓

匿名点击（X-Bond）⭐

**参与机构：**现券市场各类交易活跃机构

交易品种：集中银行间市场最活跃的债券

**交易时间：**每个交易日上、下午两个连续报价成交交易场次；连续报价成交阶段（9:00-12:00、13:30-17:00）；集中报价和集中匹配阶段可全天设定

**授信：**基于管理人层面的授信机制，X-Bond和请求报价只需维护一次。

**交易流程：**订单匿名自动匹配/未匹配订单可供点击（授信范围内成交）

**交易规则：**匿名报价，价格优先、时间优先，匹配后具名

优势：所见即可得，减少寻找对手方、询价的时间成本

**交易策略：**多个订单可设置OCO和IfDone

债券借贷市场概述⭐⭐

**定义：**债券融入方以一定数量的债券为质物，从债券融出方借入标的债券，同时约定在未来某一日期归还所借入标的债券，并由债券融出方返还相应质物的债券融通行为

期限：最长为365天

结算方式：首期结算方式为券券对付，到期结算方式为券费对付；(到期结算方式又称券券费对付，双方持券和资金三者类似现券DVP对付)

市场功能：

  • 以信用债融利率债，提高抵押品质量
  • 出售融入债券，提供有效的做空手段
  • 提升市场流动性，满足临时结算需求
  • 融出方盘活债券存量，提高投资收益

双边清算和集中清算是两种不同的资金结算方式，主要应用于证券交易市场。

双边清算（BilateralClearing）是指在交易双方之间直接进行结算，买方直接向卖方支付资金，卖方交付证券。这种结算方式下，资金和证券直接在交易对手之间流动，清算风险主要由交易对手承担。

集中清算（CentralClearing）则是指通过中央交易所或结算机构进行结算，买方和卖方的交易都与该机构直接联系。在这种模式下，买方和卖方都将资金和证券分别存放在中央结算机构，由该机构进行结算和清算。这种方式可以降低交易对手方面的清算风险，同时也提供了更高的市场流动性。

两者的主要区别在于资金的流动路径和清算风险的承担方式。双边清算强调交易对手之间的直接结算，而集中清算则借助中央机构作为中介进行结算，以降低风险并提高市场效率。

R007-2.3%，10年期国开债票面利率为3.8%，FR007_1Y为2.8%

交易员以R007滚动融入1亿元资金，购买1亿元10年国开债。

交易员同时做一笔FR007_1Y利率互换，收取浮动利率FR007，支付固定利率2.8%。

经此交易安排后，将融资成本基本固定在2.8%，实现无风险利差100BP，未来一年可获得净票息（Carry）100万元

Carry&CapitalGain：骑乘策略⭐

前提：收益率曲线向上倾斜且预期持有期内形态稳定

基本操作：购入长期限债券，在债券到期前平仓退出

风险：持有期内收益率曲线大幅上移

案例3：

国开债收益率曲线：3年-3.0%、4年-3.5%、5年-4%，R007-2%，5年期国开债票面利率-4.5%
交易员买入1亿元待偿期为5年的国开债，1年后卖出。假设R007不变，Carry=10000×（4.5%-2%）=250万
假设收益率曲线形状不变，平均久期4年，1年后债券净价上涨（4%-3.5%）×4=2%，CapitalGain=10000*2%=200万

期差交易（SpreadTrade)☆

p原理

Ø利用收益率曲线形态的预期变化进行交易

Ø本质是交易不同期限利率之间的利差

Ø总的PV01通常维持在零

p具体操作

Ø陡峭策略：预期收益率曲线变陡，卖长买短

Ø平缓策略：预期收益率曲线变平坦，卖短买长

Ø正蝶式策略：预期收益率曲线变下凸，买中间卖两边

Ø反蝶式策略：预期收益率曲线变上凸，买两边卖中间

p案例4：

当前国债收益率曲线：1年期-2.5%，10年期-3.0%

交易员预期经济将逐步复苏，国债收益率曲线斜率将变陡

买入1亿元1年期国债，卖出（做空）1000万元10年期国债

1年后，国债收益率曲线恰如预期变陡，1年期-2.0%，10年期-

3.5%

交易员卖出1亿元1年期国债，买入1000万元10年期国债

期差收益：（2.5%-2%）*10000+（3.5%-3%）101000=100

万元

利率互换期差交易☆

优势

Ø较现券易于做空

Ø可直接交易Spread

p案例5：

Ø当前FR007_1Y*5Y报价：28/22（Offer/Bid）

Ø交易员预期经济将逐步复苏，收益率曲线将变陡，Spread将扩大

Ø买入Spread，即pay1亿元FR007_5Y，sell5亿元FR007_1Y

Ø1年后，收益率曲线恰如预期变陡，FR007_1Y*5Y报价：80/78

Ø交易员卖出Spread，即Sell1亿元FR007_5Y，pay5亿元FR007_1Y

Ø期差收益：头寸价值增长（78BP-28BP）510000=250万元

l问题:RollDown（风险无法完全对冲）

基差交易（BasisTrade)☆

原理：利用不同市场之间的利差进行交易

具体操作

  • 期现基差策略：交易国债期货与现货价格的利差

  • 信用基差策略：交易不同信用等级的债券之间的利差

  • 利率互换基差策略：交易不同参考基准的利率互换之间的利差，如Shibor*FR007

  • 现券+利率互换基差策略：交易现券收益率与互换利率之间的利差

现券+利率互换基差交易☆

案例6：

当前5年期国开债与FR007_5Y的利差为120BP

交易员认为当前利差显著高估，预期利差将大幅收窄

卖出Spread，即买入1亿元5年期国开债，同时pay1亿元FR007_5Y

1年后，5年期国开债与FR007_5Y的利差如预期收窄为40BP

买入Spread，即卖出1亿元5年期国开债，同时Sell1亿元FR007_5Y

基差收益：（120BP-40BP）510000=400万元
