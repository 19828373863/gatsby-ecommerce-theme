19.1财务业绩评价

Created2024-02-21|Updated2024-04-28|CPA财务管理19.业绩评价
|Word Count:678|Reading Time:2mins

本章是比较重点的章节，主要介绍财务业绩评价与非财务业绩评价、关键绩效指标法（KPI）、经济增加（EVA）以及平衡计分卡（BSC）的知识，内容与责任会计、财务报表分析与财务预测、资本成本计算等章节有一定联系。考试形式以客观题为主，也会涉及主观题。

内容大纲

第一节财务业绩评价与非

财务业绩评价的优点与缺点
概念 财务业绩评价是根据财务信息来评价管理者业绩的方法
评价指标 净利润、资产报酬率、经济增加值等
【注意】各责任中心的业绩评价指标均是财务业绩评价
优点 1.可以反映企业综合成果
2.容易从会计系统中获得相应的数据，操作简便，易于理解，被广泛使用
缺点 1.财务业绩反映短期业绩，无法反映管理者在企业的长期业绩改善方面所作的努力
2.财务业绩是一种结果导向，只注重最终的财务结果，对达成该结果的改善过程欠考虑
3.财务业绩通过会计程序产生的会计数据进行考核，受到稳健性原则有偏估计的影响，可能无法公允地反映管理层的真正业绩

非财务业绩评价的优点与缺点
概念 非财务业绩评价，是指根据非财务信息指标来评价管理者业绩的方法
评价指标 1.与顾客相关的指标：市场份额、关键客户订货量、顾客满意度、顾客忠诚度等
2.与企业内部营运相关的指标：及时送货率、存货周转率、产品和服务的质量、周转时间等
3.与员工学习与成长相关的指标：员工满意度、员工建议次数、员工拥有并熟练使用电脑比率、员工第二专长人数、员工流动率等
优点 1.可以避免财务业绩评价只侧重过去、比较短视的不足
2.更体现企业长远业绩和外部对企业的整体评价
缺点 指标往往比较主观，数据收集困难，评价指标数据的可靠性难以保证
