稀释每股收益究竟有多难理解？

本文摘自《大白话系列之 每股收益 》每次学到稀释每股收益的计算这里时，都很抓狂，有木有？认股权证、股份期权、限制性股票、企业承诺回购股份……

一会儿认股权证的行权价格低于当期普通股平均市场价格时，才具有稀释性。

一会儿企业承诺回购的价格高于当期普通股平均市场价格时，才具有稀释性。

后面还有限制性股票的现金股利可撤销和不可撤销时每股收益的计算……被绕的云里雾里的。

其实，不用什么大原理，说通了，真的真的很简单。

先简单说下，下面再细说如何理解和计算：

认股权证，它的行权价格是持有人去购买企业的股票时支付的价格，企业是收款方。企业收到每股价格（持有人的行权价）低于市场价格，必然会稀释企业的每股收益。把不甜的糖开水倒入甜的糖开水，甜的糖开水肯定没有以前甜了。

企业承诺会回购股东手里的股份，这时企业是付款方，必然是回购价格高于当期普通股平均市场价格，才具有稀释性。把一件苹果中比平均重量大的苹果给了别人，剩下的苹果的平均重量肯定会下降。

具体计算的理解和计算，通过一道简单的例题来解释：

某公司20×7年度归属于普通股股东的净利润为2750万元，发行在外普通股加权平均数为5000万股，该普通股平均每股市场价格为8元。20×7年1月1日，该公司对外发行1000万份认股权证，行权日为20×7年3月1日，每份认股权证可以在行权日以7元的价格认购本公司1股新发的股份。

这种情况就是上面说的，行权价格低于市场价格，此时持有人才可能选择行权。

企业的基本每股收益=2750/5000=0.55元/股

也就是，公司现在的5000万股，价格是每股8元，对应的每股收益是0.55元，如果增加1000万股价格为7元的股票，对应的每股收益就会下降。如上说的，把不甜的糖开水倒入甜的糖开水，甜的糖开水肯定就没有以前甜了，因为被稀释了，甜度降低。

所以，这时持有人以低于7元的价格来买公司的1000万股股票，对公司每股收益具有稀释性。

确定了具有稀释性，计算稀释每股收益时才需要考虑，否则不需要考虑。

在计算稀释每股收益时，一般情况不考虑认股权证或股份期权对净利润的影响，所以只需要考虑调增的普通股股数。

因为现在8元的价格对应的是0.55元的每股收益，如果再持有人以8元的市场价购买公司股票，就对每股收益既没有稀释性，也没有反稀释性。把两杯甜度相同的糖开水倒在一起，甜度不会变化。

但现在是持有人以7元低于市场价格的价格购买的公司股票1000万股，7元的价格就支撑不起0.55元的每股收益的每股收益，具有稀释性。

现在是要计算这个稀释性究竟有多大（就是计算稀释每股收益），就要想办法把7元购买1000万股换算成用8元的价格购买的股份数，

持有人行权时，公司收到的钱=7*1000=7000万元。假如公司以市场价格8元发行新股（即持有人以8元的价格购买股票），要收到同样的7000万元，那就需要发行7000/8=875万股。

也就是说，如果企业以8元的市场价格发行875万股，企业能收到7000万元，但这875万股对企业没有稀释性。

现在的实际情况是，企业多了1000万股股票，收到了7000万元。就相当于这1000万股中，有875万股没有稀释性，剩余的1000-7000/8=125万股，是企业白送给持有人的，持有人没支付任何对价，企业一分钱没收到，只是纯粹增加125万股股份，就是净增加。

用前面的增量每股收益来计算，增量每股收益=假设转股增加的净利润/假设转股增加的普通股股数的加权平均数=0 / 净增加的股份数 = 0。0当然小于基本每股收益0.55元，必然具有稀释性。

因此，稀释每股收益 = 净利润 /（原来的普通股+净增加的普通股）

= 2750/（5000+125）=0.54元/股

最终算出来的稀释每股收益为0.54元 < 基本每股收益0.55元，如果都大于或等于了，就证明判断或计算错误了，稀释一定是变小。

通过这道题目，总结一下认股权证或股份期权的稀释每股收益的计算。

对于盈利企业，认股权证和股份期权等的行权价格低于当期普通股平均市场价格时，具有其稀释性。

净增加的普通股股数 = 行权时转换的普通股股数-行权价格×行权时转换的普通股股数 / 当期普通股平均市场价格

稀释每股收益 = 净利润 /（原普通股的加权平均数 + 净增加的普通股股数的加权平均数）
