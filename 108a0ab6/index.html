分期购买固定资产，如何处理和计算？

分期购买固定资产和我们日常生活中所说的分期付款是一回事，比如分期购买手机、汽车、房子。

选择分期付款的原因是一次性付款金额太大了，虽然分期付款的总金额会比一次性付款的金额多一些，但每期还款压力更小。

分期付款的总金额与一次性付款的金额之间的差额其实就是利息，分期付款的总金额就包括了本金和利息。所以，分期付款购买固定资产实际上是具有融资性质，相当于向卖家借钱来买固定资产，然后分期向卖家还款，类似按揭买房。

这种情况下，买进固定资产的成本应该按照分期付款总金额的现值进行计算，就是本金部分，而不是利用分期付款的总金额进行计算，总金额包括了利息，扣除利息后的本金部分才对应的是固定资产的现在的真实成本。

分期付款买进固定资产的会计分录为：

借:在建工程/固定资产 (分期付款总额的现值)

未确认融资费用 (应付利息)

贷:长期应付款 (分期付款的总额)

分期付款需要支付的总金额对应的会计科目是长期应付款，就是超过正常信用条件的分期付款，超过正常信用条件的意思就是分期需要支付利息，正常信用条件下的分期是不需要支付利息的，比如只分三两期付款，可能就不需要支付利息。

长期应付款包括本金和利息，其中利息对应的科目是叫未确认融资费用，因为分期付款是未来才支付，利息费用每期才会实际发生，实际发生了每期末才确认，所以没有发生以前都叫未确认，还没有确认的因为融资产生的费用，就是未确认融资费用。

未确认融资费用是长期应付款的备抵科目，长期应付款是负债类科目，借减贷增，与之相反，未确认融资费用就是借增贷减。

会计中为什么要设置备抵类科目，直接从成本里面扣除不方便吗？

1、首先在会计理论和方法中会计计量占有很重要的地位，它分为初始计量和后续计量。比如你买了部手机，买的时候花了多少钱？用了一段时间现在值多少钱？显然这两个数据是不一致的。这种现象的产生导致了会计计量属性的约定：历史成本、重置成本、可变现净值、公允价值、未来现金流量现值。为了直观对比这种数据差异的存在就产生了会计备抵核算的方法，备抵核算采用“备抵账户”核算。比如各种减值准备账户、累计折旧、未实现融资收益、未确认融资费用、利润分配这些都是属于备抵类账户。

2、那么为什么账面还要保留原始成本呢？举个栗子，你的一个亲戚借了你10万元做生意赔了。本来说好的一个月就还本金，经过了半年他根本不再提借钱这回事儿，你问他要，他说还不上了，自己的家产就是一个破电动车，你看看了电动车最多值1000元，住房是租的，其他家徒四壁。基于这种情况，你就提取9.9万的坏账准备吧。账面原值还是10万，净值只有1000元。你一气之下把你这个不靠谱的亲戚起诉到了法院，法官问你借了多少，你回答借了1000元？显然这不是事情的本来面目。你还是告诉法官他借了你10万对吧。

概念理解：未确认融资费用相当于长期应付款的备抵科目。

这个科目存在的意义是什么呢？

我们知道 长期应付款代表的是未来要留出现金的总额，是反映未来一系列时间的事情。而资产负债表反映的是资产负债表日当天 资产或负债的情况，所以长期应付款的列报金额也应该反映在资产负债表日这个时间点上长期应付款的情况，显然未来的现金流不可以反映当前时点的情况，当前时点的情况应该用未来流出现金流的现值反映，及长期应付款的账面价值=长期应付款的现值。

长期应付款的账面余额：未来流出现金流总额

长期应付款的账面价值：未来现金流的现值

长期应付款的账面价值=长期应付款账面余额-未确认融资费用

即：未确认融资费用 =长期应付款未来现金流总额（账面余额）-长期应付款资产负债表日的现值

相当于：本金是长期应付款的现值，未来要付的利息（承担的资金时间成本）就是未确认融资费用。

再通俗一点，分十年花200万买一个房子，未来十年的现金流折现=150万，200-150=50就是未确认融资费用，是资金的时间成本。未确认表示以后要确认，相当于借了一年钱就确认一年利息。

借：房子 150万

​ 未确认融资费用 50万

贷：长期应付款 200万

以后每经过一年，确认一笔融资费用

借：财务费用

贷：未确认融资费用（长期应付款账面价值*折现率）

理解：长期应付款账面价值=长期应付款的现值≈本金 折现率≈利率 即，借了人家一年的本金，确认一笔占用资金的利息

总结：两个理解的关键点

1.资产负债表反映当前，长期应付款这个科目反映未来一段时间，所以要用 “未确认融资费用”将账面余额调整至列报金额 用现值 反映 长期应付款在这个时间点的情况。

2.考虑资金的时间价值

以上讨论假设长期应付款不存在一年内到期的情况。

既然分期付款的总额（长期应付款）包括本金和利息，那购买以后每期（可能是每年或每半年）分期付款的金额中也应该相应的分摊了本金和利息，具体是多少，如何计算？

这个计算就是运用之前将的摊余成本法，就是需要先计算出每期的实际本金（每期的摊余成本），然后再根据本金计算出当期的利息。

借款的利息费用一般是计入财务费用这个科目，属于利润表中的费用类科目。但在固定资产的建造过程中或在买进需要安装的固定资产（就是对应的在建工程期间），这个期间产生的利息费用如果符合条件是应该直接计入固定资产成本的，也就是直接计入在建工程，而不是财务费用。

这种计入资产成本的利息费用叫做符合资本化条件的利息费用，与之对应的直接计入财务费用的利息费用叫做符合费用化条件的利息费用。

每期分期付款时的会计分录为：

借：长期应付款 （每期的付款金额）

贷：银行存款

借：财务费用/在建工程

贷：未确认融资费用

利息费用符合资本化条件的，计入在建工程；符合费用化条件的计入财务费用。

每期确认一部分利息费用，也就是确认一部分融资费用，相应的未确认的利息费用就减少了，所以，未确认融资费用就在贷方，表示减少。

具体金额的计算，我们通过一道教材例题来学习。

2x15年1月1日，甲公司与乙公司签订一项购货合同，甲公司从乙公司购入一台需要安装的特大型设备。合同约定，甲公司采用分期付款方式支付价款。该设备价款共计900万元(不考虑增值税)，在2x15年至2x19年的5年内每半年支付90万元，每年的付款日期分别为当年6
月 3 0 日和12月31日。2x15年1月1日，设备如期运抵甲公司并开始安装。2x15年12月3
1日，设备达到预定可使用状态，发生安装费398530.60元 ，已用银行存款付讫。不考虑相关税费的影响。假定甲公司适用的6个月折现率为10%。

( 1 ) 固定资产的成本为分期付款的现值：

900000 *(P/A ,10%,10 ) =900000 *6.1446=5530140 元

这个系数(P/A ,10%,10 )=6.1446，不需要自己计算，考试时会给系数表，直接查表可得。

2x15年1月1日甲公司的账务处理如下:

借：在建工程 5530140

未确认融资费用 3469860 （9000000-5530140）

贷：长期应付款 9000000 （5290万）

(2 ) 每期分期付款时，各种金额的计算：（我直接画个表一目了然）

确认的融资费用=期初本金*10%。

每期支付的本金=900000-当期确认的的融资费用。

剩余的应付本金=上一期剩余的应付本金-本期已支付的本金。

剩余的未确认融资费用=上一期剩余的未确认融资费用-本期确认的融资费用。

由此可以总结出：

剩余的应付本金=上一期剩余的应付本金-本期已支付的本金

=上一期剩余的应付本金-（900000-当期确认的的融资费用）

=上一期剩余的应付本金-900000+当期确认的的融资费用

=上一期剩余的应付本金-900000+期初本金*10%

=期初本金*（1+10%）-900000

最后一步可能看不懂，解释一下：

上一期剩余的应付本金就是上一期期末应付的本金，就是本期期初应付的本金。

所以：

剩余的应付本金=期末应付的本金=期初本金*（1+10%）-900000

这就是摊余成本的计算方式：

某一期的期末摊余成本=期初摊余成本*（1+实际利率）-支付的钱

某一期的利息费用=期初摊余成本*实际利率

根据表中的计算，我们来写会计分录。

15年6月30日，第一次付款：

借：长期应付款 900000

贷：银行存款 900000

借：在建工程 553014

贷：未确认融资费用 553014

这一期的利息费用=期初摊余成本5530140*实际利率10%=553014。

这次确认的利息费用的期间是15年1月1日到6月30日的，属于安装期间。因为是买入需要安装的固定资产，安装期间的利息费用应该资本化，直接计入在建工程。

15年12月31日，第二次付款：

借：长期应付款 900000

贷：银行存款 900000

借：在建工程 518315.4

贷：未确认融资费用 518315.4

这一期的期初摊余成本=5530140*(1+10%)-900000=5183154元，利息费用=期初摊余成本实际利率=(5530140(1+10%)-900000)*10%=518315.4元。

这次确认的利息费用的期间是15年7月1日到12月31日的，还属于安装期间，利息费用应该资本化，直接计入在建工程。

由于题目中说， 15年12月3 1日，设备达到预定可使用状态，意思就是安装期间结束。根据前面学的，这时需要从在建工程转入固定资产了。

安装期间除了发生了利息费用，还发生了安装费用，都应该计入在建工程。

安装费用的会计分录：

借：在建工程 398530.6

贷：银行存款 398530.6

在建工程转入固定资产的会计分录：

借：固定资产 7000000

贷：在建工程 7000000

在建工程的总余额=5530140+553014+518315.4+398530.6=7000000元，全部转入固定资产，固定资产的初始初始成本就是7000000元。

安装虽然结束了，分期付款还没付完，还得继续付款。

16年6月30日，第三次付款：

借：长期应付款 900000

贷：银行存款 900000

借：财务费用 480146.94

贷：未确认融资费用 480146.94

这一期的期初摊余成本=5183154*(1+10%)-900000=4801469.4元，利息费用=期初摊余成本实际利率=(5183154(1+10%)-900000)*10%=480146.94元。

这次确认的利息费用的期间是15年12月31日到16年6月30日的，已经安装结束，从此不再属于安装期间，利息费用应该费用化，直接计入财务费用。
